Create a Docker Hub account
    Go to https://hub.docker.com/. Sign up. Give your username to Alexis (via Skype, for example).

Install Docker
    Mac
        Paste the following URL into a browser: https://download.docker.com/mac/stable/Docker.dmg.
        
        Double-click the DMG file and follow the instructions to install Docker.
        
    Ubuntu
        There is an excellent guide at https://docs.docker.com/engine/installation/linux/docker-ce/ubuntu/.

Start a VA-PALS Docker container
    Mac
        Open the app Terminal.
        
    Ubuntu
        Start a terminal session.
    
    At the shell prompt, enter the following:
    
    $ docker pull vistaexpertisenetwork/va-pals
    $ docker run -d --name avicenna -h avicenna -p 2222:22 -p 8001:8001 -p 8080:8080 -p 8888:8888 -p 9430:9430 -p 9080:9080 -P --entrypoint "/bin/sh" vistaexpertisenetwork/va-pals -c /home/osehra/bin/start.sh
    
Start Vista
    Technically, Vista has already started, but anyhoo...
    
    $ ssh -p 2222 root@localhost
    root@localhost's password: docker
    # osehra (the '#' is the root user's prompt)
    $ mumps -dir
    
Stop the container
    $ exit
    # exit
    Back at your local shell prompt:
    $ docker stop avicenna
    
    To start it back up:
    
    $ docker start avicenna
    
For slightly more general information about your VA-PALS container, see https://hub.docker.com/r/vistaexpertisenetwork/va-pals/.
